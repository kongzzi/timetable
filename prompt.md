# 초기 프롬프트

시간표 작성 앱을 만들고 싶어.

[무엇을] 코딩 경험이 없는 초보자용으로, ‘대학생 시간표’ 웹페이지를 만들어줘.
[조건] 파일은 index.html 과 style.css, script.js로 나눠줘. 외부 라이브러리는 쓰지 말아줘.
[스타일] 색깔은 파스텔톤으로 부드럽고 밝고 귀여운 스타일로 해 줘.
[확인] 브라우저에서 index.html을 열면 바로 보이게 해줘.
[범위] my_card 폴더 안의 파일을 수정해서 만들어줘.
[마무리] 실행 방법(어떤 파일을 열면 되는지)과 수정 포인트 3가지를 알려줘.

[시간표 내용]
- 1교시는 9시이고 1시간 단위로 8교시까지 만들어 줘.
- 30분 단위로 입력, 시간표에 표시도 되게 해줘.
- 시간표 옆에 강의명, 강의실, 수업시간등을 입력하는 폼도 만들어 줘.

# 1차수정
제목을 "내 대학 시간표"가 아닌 "2026년 Kongzzi의 1학기 시간표"로 고쳐줘. index.html만 수정해줘.
